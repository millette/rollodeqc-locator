# rollodeqc-locator
> Parse locations to extract city, etc.

## Install
```
$ npm install --save rollodeqc-locator
```

## Usage
```js
const rollodeqcLocator = require('rollodeqc-locator');

rollodeqcLocator('unicorns');
//=> 'unicorns & rainbows'
```
